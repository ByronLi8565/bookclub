Note avatars

Top card is how notes look today. Below are ways to fold a user's profile picture in, all using the existing square / black-border avatar (initials as fallback) so nothing fights the design language.

Current — no avatar

Seq number box, a title line ("author posted / replied <time>"), then the body. Actions live on the right of the head.
